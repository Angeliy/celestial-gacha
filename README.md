# Celestial Gacha

Celestial Gacha 是一个基于 Stitch 设计稿 `Oriental Ink Gacha Simulator` 实现的国风水墨抽卡模拟器。

## 功能

- 祈愿池首页
- 抽卡页面
- 统计页面
- 角色图鉴
- SSR 基础概率 1.6%
- SSR 90 抽保底
- 单抽、十连、保底重置、抽卡记录、图鉴收集联动
- 所有主要交互元素均配置 `data-testid`

## 技术栈

- React
- Vite
- TypeScript
- Tailwind CSS
- Vitest

## 本地运行

```bash
npm install
npm run dev
```

默认访问：

```text
http://127.0.0.1:5173/
```

## 测试

```bash
npm run test
npm run build
npm run test:ci
```
